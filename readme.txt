Add Custom Menus with a shortcode: [custommenu menu=Menu].

== Description ==

This plugin allows you to display your WordPress Custom Menu in a Page, Post or other Custom Post Type. Easily by adding a shortcode.

== Installation ==

1. Upload the custom-menu-shortcode folder to the /wp-content/plugins/ directory
Activate the Custom Menu Shortcode plugin through the 'Plugins' menu in WordPress
Display your Custom Menus with a shortcode: `[custommenu menu=Menu-name-here].`

== Frequently Asked Questions ==

= How do I create menus? =

This plugin just helps you to show Custom Menus you've already created in "Appearance > Menus".
